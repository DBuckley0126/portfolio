# OutlineSquareButton

A general use button built ontop of **Material UI button** being animated with Framer Motion.
When focused created a gradient effect across the outline and text.
Font size automatically resizes based on device.

Made to Web Content Accessibility Guidelines.

## Libaries

-   [Material UI](https://mui.com/getting-started/usage/)
-   [Framer Motion](https://www.framer.com/docs/)
