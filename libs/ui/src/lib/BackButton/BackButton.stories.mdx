# BackButton

A back navigation button built ontop of **Material UI button** being animated with Framer Motion.
When hovered applies a backwards motion with the arrow.

Made to Web Content Accessibility Guidelines with customised focus styling.

## Libaries

-   [Material UI](https://mui.com/getting-started/usage/)
-   [Framer Motion](https://www.framer.com/docs/)
